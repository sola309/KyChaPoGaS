## B. 操作性 — 今回修正済み ✅

1. **再生追従**: 再生中プレイヘッドが画面外に出たら自動スクロール(手動閲覧には介入しない条件付き)
2. **ドラッグスクラブ**: ルーラーを押しっぱなしで現在位置を連続調整(PointerEventsなので**タッチでも動く**)
3. **Space**で再生/停止(入力欄フォーカス時は無効)
4. **←/→でコマ送り**(Shiftで±10フレーム)
5. **カーソル位置基準のズーム**: Ctrl+ホイールでカーソル下のフレームが画面上で動かない
6. **スマホ対応**: 構成/音楽/LoRAタブとショットエディタを縦積みレイアウト化(lg未満で自動切替)
